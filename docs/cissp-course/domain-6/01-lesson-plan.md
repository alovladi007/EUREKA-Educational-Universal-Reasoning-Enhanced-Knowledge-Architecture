## Summary

Domain 6 covers assessment strategies, testing techniques (vulnerability
assessment, penetration testing, fuzzing, code review, SAST/DAST/IAST),
audit management, log review, SOC reports, and collecting data for security
management. Mantra: **"Evidence over opinion; independence matters."**
