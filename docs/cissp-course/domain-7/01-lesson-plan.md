## Summary

Domain 7 is the "day two" domain — what you do after the security architecture
is built. Covers investigations, logging and monitoring, configuration and change
management, patch and vulnerability management, incident response, disaster
recovery, and physical security operations. Mantra: **"Detect, respond, recover, improve."**
