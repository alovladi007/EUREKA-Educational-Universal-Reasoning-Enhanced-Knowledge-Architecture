# EUREKA Web Application

Next.js 14 web portal for learners and educators.

## Features
- Learner dashboard
- Educator portal
- Course management
- Assessment interface
- Real-time collaboration
- WCAG 2.2 AA accessible

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui components
- next-intl (i18n)
