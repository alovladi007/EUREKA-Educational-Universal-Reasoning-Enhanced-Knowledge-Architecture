# Solar System Textures — Credits

The planetary surface, cloud, atmosphere, ring, and Milky Way background
textures in this directory are from **Solar System Scope**
(https://www.solarsystemscope.com/textures/).

## Original data sources

The textures themselves are derived from NASA planetary mission imagery
and elevation data — Voyager, Cassini, Magellan, MESSENGER, MRO, etc.
NASA mission data is in the U.S. public domain. Solar System Scope's
contribution is the equirectangular-projection processing and
consistent visual styling.

## Files

  2k_sun.jpg                  — Sun surface (composite of SDO / SOHO data)
  2k_mercury.jpg              — Mercury (MESSENGER mission)
  2k_venus_surface.jpg        — Venus surface (Magellan radar)
  2k_venus_atmosphere.jpg     — Venus atmosphere (Galileo, Mariner)
  2k_earth_daymap.jpg         — Earth day side (Blue Marble Next Generation)
  2k_earth_nightmap.jpg       — Earth night lights (Black Marble / VIIRS)
  2k_earth_clouds.jpg         — Earth cloud layer (NASA composite)
  2k_mars.jpg                 — Mars (MOLA elevation + MRO surface)
  2k_jupiter.jpg              — Jupiter (Voyager, Cassini, JunoCam)
  2k_saturn.jpg               — Saturn (Voyager, Cassini)

  2k_uranus.jpg               — Uranus (Voyager 2)
