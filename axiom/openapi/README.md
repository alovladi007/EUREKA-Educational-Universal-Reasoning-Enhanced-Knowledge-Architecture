# API contract

The OpenAPI contract is generated by the FastAPI app and served at
`GET /openapi.json`. To snapshot it for contract tests or client generation:

    curl http://localhost:8400/openapi.json > openapi/axiom.v1.json

Phase 0 defines the identity and dashboard surface (GET /api/v1/me, GET
/api/v1/dashboard/summary) plus the ops probes (/health, /ready, /metrics).
As each domain lands, its routes are added under /api/v1 and the snapshot is
refreshed. Contract tests assert the served spec matches the committed
snapshot so a breaking change is caught in CI.
