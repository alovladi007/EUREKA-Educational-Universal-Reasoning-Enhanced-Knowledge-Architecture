## 🧪 Quick Start Commands

### Start the Service
```bash
cd eureka
docker-compose up -d test-prep
```

### View Logs
```bash
docker logs -f eureka-test-prep
```

### Initialize Database
```bash
docker exec -it eureka-test-prep python seed_database.py
```

### Run Tests
```bash
docker exec -it eureka-test-prep pytest tests/ -v
```

### Stop the Service
```bash
docker-compose stop test-prep
```
